# Fylo Website

Fylo landing page with dark/light mode switcher using Tailwindcss from [Frontend Mentor Challenge](https://www.frontendmentor.io/challenges/fylo-dark-theme-landing-page-5ca5f2d21e82137ec91a50fd)

## Usage

Install dependencies

```
npm Install
```

Run Tailwind CLI

```
npm run watch
```
